# Windows 智能关机助手
基于HTML+Node.js开发的Windows自动关机程序，支持定时关机、每日/每月重复、即时关机等功能，界面精美，兼容所有Windows系统。

## 功能特点
- ✅ 支持单次/每日/每月定时关机
- ✅ 实时倒计时显示
- ✅ 一键取消关机/立即关机
- ✅ 精美Web界面，操作简单
- ✅ 兼容Windows 7/8/10/11

## 快速开始
### 1. 环境准备
安装Node.js（v14+）：https://nodejs.org/
